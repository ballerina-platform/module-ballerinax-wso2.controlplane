## Overview

The WSO2 Control Plane connector provides internal support for exposing runtime artifacts to the [WSO2 Integration Control Plane](https://wso2.com/).

### Key Features

- Expose Ballerina runtime artifacts to the integration control plane
- Automatic service discovery and registration
- Seamless integration with WSO2 integration ecosystem
